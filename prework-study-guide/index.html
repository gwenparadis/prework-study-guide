Prework Study Guide

HTML

  • HTML stands for Hypertext Markup Language
  • HTML is the first stage in the webpage development process
  • HTML creates the frame for the webpage, by organizing the content of the webpage
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The most comment elements included in HTML code are: text elements, image elements, header elements, and list elements
  • These elements are often referred to as parent elements, as they usually include additional child elements (such as a paragraph of text (CE) inside the body of the page (PE))

CSS

  • CSS stands for Cascading Style Sheets
  • CSS is the second stage in the webpage development process
  • CSS adds style to the basic HTML frame created in the fist stage of webpage development
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS style files are made separately from the HTML framework, and linked by adding the link in the HTML metadata

GIT

  • git status: confirms which git we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

Javascript

  • Javascript (JS) is a coding language used to create interactive elements on a webpage
